SITE HAVR TECNOLOGIA
===================

Como fazer o deploy:
1. Copie todos os arquivos para seu servidor web
2. Certifique-se de que os arquivos estão na raiz do domínio
3. O site irá funcionar imediatamente!

Arquivos necessários:
- index.html (página principal)
- styles.css (estilos)
- script.js (funcionalidades)
- attached_assets/ (pasta com todas as imagens)

Funcionalidades incluídas:
✓ Site totalmente responsivo
✓ Menu mobile
✓ Formulário de contato funcional
✓ Botões WhatsApp integrados
✓ Animações suaves
✓ SEO otimizado
✓ Compatível com todos os navegadores

Tecnologias usadas:
- HTML5 puro
- CSS3 com design responsivo
- JavaScript vanilla (sem dependências)
- Ícones via Lucide CDN

Para personalizar:
- Edite as cores no arquivo styles.css (seção :root)
- Troque as imagens na pasta attached_assets/
- Modifique textos diretamente no index.html
- Ajuste número do WhatsApp nos links

Compatibilidade:
✓ Chrome, Firefox, Safari, Edge
✓ Dispositivos móveis e tablets
✓ Velocidade otimizada
✓ Acessibilidade incluída
